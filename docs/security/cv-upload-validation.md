# التحقق من رفع السيرة

## القرار الأمني

**PDF فقط** — لا يوجد Malware Scanner فعلي؛ DOC/DOCX غير مقبولين.

## الفحوصات

1. حجم أقصى: `CV_MAX_SIZE_KB` (افتراضي 10240)
2. امتداد `pdf` فقط
3. MIME عبر `finfo` → `application/pdf`
4. Magic bytes: `%PDF-`
5. منع الامتداد المزدوج
6. إعادة تسمية داخلية (UUID)
7. تخزين على قرص خاص غير قابل للتنفيذ
8. تنزيل كـ attachment دائماً

## فشل الرفع

- orphan file يُحذف إذا فشل commit قاعدة البيانات.
