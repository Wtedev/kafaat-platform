# تدفق بيانات قاعدة المرشحين

```mermaid
flowchart LR
    U[المستخدم] -->|رفع PDF| PV[تخزين خاص للسيرة]
    U -->|موافقة صريحة| CP[تفضيلات قاعدة المرشحين]
    CP -->|granted + CV + ملف مكتمل| PL[قائمة المرشحين الداخلية]
    PL -->|عرض / تنزيل| ST[موظف مخوّل]
    ST --> AL[Audit Log]
    PV --> AL
    CP --> AL
```

لا توجد جهة خارجية في هذا المخطط.
