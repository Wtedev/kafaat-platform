# مركز الخصوصية (بوابة المستفيد)

## المسار

- `GET /portal/privacy` — الاسم في الواجهة: **الخصوصية والبيانات**

## من يمكنه الوصول

- مستخدم مسجّل واجتاز OTP في الجلسة الحالية.
- حسابات `deletion_pending` مسموح لها ما دام التنفيذ النهائي لم يبدأ.
- **محظور:** حسابات `anonymized` و`deletion_processing`.

## أقسام الصفحة

| القسم | ما يُعرض |
| --- | --- |
| بيانات الحساب | الاسم الرباعي، البريد، الجوال، تاريخ الميلاد، نوع الهوية، الهوية **مقنّعة**، تواريخ الإنشاء/التحديث، حالة الحساب |
| سياسة الخصوصية | الإصدار الفعّال، آخر إقرار، تاريخ الإقرار، حاجة إعادة إقرار، روابط السياسة |
| قاعدة المرشحين | الحالة، تاريخ آخر قرار، إصدار النص، رابط الإعدادات |
| السيرة الذاتية | وجود CV، تاريخ الرفع، النوع، الحجم، آخر تحديث، تنزيل/حذف آمن |
| طلبات الخصوصية | UUID، النوع، الحالة، التواريخ، رسالة القرار للمستخدم |
| الإجراءات | طلب وصول، طلب تصحيح، طلب حذف حساب (Workflow الحالي) |

## ما لا يُعرض

- الهوية الكاملة، ciphertext، lookup hash، password hash، remember token.
- مسارات الملفات، checksum، disk، OTP، Security Logs، ملاحظات داخلية، `assigned_to`.
- زر تصدير ZIP (خارج نطاق المرحلة الحالية).

## التنفيذ

- ViewModel: `App\Data\Privacy\PortalPrivacyCenterViewData` (Allowlist).
- Controller: `PortalPrivacyCenterController`.
