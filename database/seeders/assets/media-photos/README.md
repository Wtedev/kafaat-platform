# أصول صور المركز الإعلامي

هذا المجلد مصدر الحقيقة لصور **المركز الإعلامي → الصور** في الموقع العام.

## الهيكل

```
media-photos/
├── الفعاليات والمبادرات/
│   └── {اسم الفعالية أو المبادرة}/
│       └── *.jpg | *.jpeg | *.png | *.webp
├── زيارات واستضافات/
│   └── {اسم الزيارة أو الاستضافة}/
└── مرافق الجمعية/
    └── {اسم المجموعة}/
```

## النشر

عند التشغيل:

```bash
php artisan db:seed --class=MediaPhotoSeeder --force
```

يُنسَخ المحتوى إلى `storage/app/public/media/photos/library/` وتُسجَّل السجلات في `media_photos`.

على Railway يُشغَّل الـ seeder تلقائياً ضمن `railway/predeploy.sh`.

## ملاحظات

- الصيغ المدعومة: JPEG و PNG و WebP فقط (ملفات RAW مثل `.CR3` تُتخطى).
- أضف صوراً جديدة داخل المجلد المناسب ثم أعد تشغيل الـ seeder.
