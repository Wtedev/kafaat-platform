# أصول صور المركز الإعلامي

هذا المجلد مصدر الحقيقة لصور **المركز الإعلامي → الصور** في الموقع العام.

## الهيكل

```
media-photos/
├── الفعاليات والمبادرات/
│   └── {اسم الفعالية أو المبادرة}/
│       └── *.jpg
├── زيارات واستضافات/
│   └── {اسم الزيارة أو الاستضافة}/
└── مرافق الجمعية/
    └── {اسم المجموعة}/
```

## التصنيف

| القسم | المحتوى |
|--------|---------|
| الفعاليات والمبادرات | فعاليات ومبادرات الجمعية المجتمعية والتدريبية |
| زيارات واستضافات | زيارات الشركاء والجهات الداعمة |
| مرافق الجمعية | صور مقر ومرافق الجمعية |

## إضافة صور جديدة

1. ضع الصور في المجلد المناسب (بدون مسافات زائدة في أسماء المجلدات).
2. شغّل ضغط الأصول:

```bash
php scripts/optimize-media-photo-assets.php
```

3. انشر السجلات:

```bash
php artisan db:seed --class=MediaPhotoSeeder --force
```

يُنسَخ المحتوى إلى `storage/app/public/media/photos/library/` وتُسجَّل السجلات في `media_photos`.

على Railway يُشغَّل الـ seeder تلقائياً ضمن `railway/predeploy.sh`.

## ملاحظات

- الصيغ المدعومة للمصدر: JPEG و PNG و WebP (تُحوَّل إلى JPEG محسّن للويب).
- ملفات RAW مثل `.CR3` تُتخطى.
- الحد الأقصى لطول الحافة بعد الضغط: 1920 بكسل.
