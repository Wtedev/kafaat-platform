# منصة كفاءات

## نبذة عن المشروع

منصة كفاءات هي منصة رقمية متكاملة مخصصة للمنظمات غير الربحية، تُتيح إدارة البرامج التدريبية والمسارات التعليمية وفرص التطوع، وتتابع تسجيل المستفيدين وموافقات الإدارة وسجلات الحضور والدرجات وساعات التطوع، وتُصدر الشهادات الرقمية تلقائياً عند استيفاء الشروط.

---

## أهم المميزات

- إدارة البرامج التدريبية والمسارات التعليمية
- تسجيل المستفيدين في البرامج والمسارات والفرص التطوعية
- الموافقة على الطلبات من لوحة التحكم الإدارية
- متابعة الحضور والدرجات لكل مستفيد
- إدارة الفرص التطوعية واعتماد الساعات
- إصدار الشهادات تلقائياً عند استيفاء شروط الاجتياز
- التحقق من صحة الشهادات عبر رابط عام
- لوحة تحكم إدارية مبنية على Filament
- واجهة مستخدم عربية بالكامل (RTL)

---

## أدوار المستخدمين

| الدور | الصلاحيات |
|---|---|
| **مدير النظام** | صلاحيات كاملة على لوحة التحكم |
| **موظف** | إدارة الطلبات والموافقات |
| **مستفيد** | التسجيل، متابعة التقدم، تحميل الشهادات |
| **زائر** | تصفح البرامج والتحقق من الشهادات |

---

## طريقة عمل المنصة بشكل مبسط

```
المستفيد يسجّل حساباً
        ↓
يستعرض البرامج التدريبية والمسارات التعليمية والفرص التطوعية
        ↓
يرسل طلب تسجيل
        ↓
الإدارة تراجع الطلب وتوافق عليه أو ترفضه
        ↓
يتابع المستفيد البرنامج / يسجّل ساعات التطوع
        ↓
الإدارة تُحدّث الحضور والدرجات / تعتمد الساعات
        ↓
عند استيفاء الشروط تُصدر الشهادة تلقائياً
        ↓
المستفيد يُحمّل الشهادة ويمكن التحقق منها عبر رابط عام
```

## متطلبات التشغيل

- **PHP** 8.2 أو أحدث
- **Composer**
- **Node.js** و **npm**
- **PostgreSQL** (أو SQLite للتطوير المحلي)
- **Laravel** 11
- إعداد خادم بريد لإرسال روابط إعادة تعيين كلمة المرور

---

## خطوات التشغيل محليًا

```bash
# 1. تثبيت الاعتماديات
composer install
npm install

# 2. إعداد ملف البيئة
cp .env.example .env
php artisan key:generate

# 3. ضبط قاعدة البيانات في ملف .env ثم تشغيل الترحيل والبذر
php artisan migrate --seed

# 4. بناء الأصول
npm run build

# 5. تشغيل الخادم
php artisan serve
```

بعد التشغيل، افتح المتصفح على: [http://localhost:8000](http://localhost:8000)

لوحة التحكم الإدارية: [http://localhost:8000/admin](http://localhost:8000/admin)

## إعداد البريد الإلكتروني

تستخدم المنصة البريد الإلكتروني لإرسال روابط إعادة تعيين كلمة المرور.

أضف بيانات خادم البريد في ملف `.env`:

```dotenv
MAIL_MAILER=smtp
MAIL_HOST=smtp.yourprovider.com
MAIL_PORT=587
MAIL_USERNAME=your@email.com
MAIL_PASSWORD=your-password
MAIL_FROM_ADDRESS=noreply@kafaat.com
MAIL_FROM_NAME="منصة كفاءات"
```

> **للتطوير المحلي:** يمكن الإبقاء على `MAIL_MAILER=log` لتسجيل الرسائل في ملف `storage/logs/laravel.log` بدلاً من إرسالها فعلياً.

## ملاحظات مهمة قبل الإنتاج

- **لا ترفع ملف `.env` إلى GitHub أبداً** — يحتوي على بيانات حساسة
- اضبط إعدادات البريد الإلكتروني بخادم SMTP حقيقي
- اضبط بيانات قاعدة البيانات الإنتاجية
- راجع صلاحيات المستخدمين من لوحة التحكم قبل الإطلاق
- اختبر إصدار الشهادات والتحقق منها في البيئة الإنتاجية
- عيّن `APP_DEBUG=false` و `APP_ENV=production` في الإنتاج
- شغّل `php artisan config:cache` و `php artisan route:cache` في الإنتاج

## حالة المشروع

المشروع في مرحلة **MVP (الحد الأدنى من المنتج القابل للتطبيق)** — جاهز للعرض التجريبي والتطوير الإضافي.

المميزات المنفذة بالكامل واجتازت اختبارات الوظائف. الخطوات القادمة المقترحة:

- توحيد نظام الأدوار (دمج `role_type` مع Spatie Permissions)
- بناء تقارير Excel باستخدام Maatwebsite Excel
- إضافة اختبارات وحدة وتكامل شاملة
- نشر على خادم إنتاج مع Horizon للطوابير
